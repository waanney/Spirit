# Spirit

A GameMaker Studio 2 adventure game project featuring exploration, dialogue systems, and mini-games.

## Overview

Spirit is a narrative-driven adventure game built with GameMaker Studio 2. The game follows a character named Henry through various locations including home, school, streets, and playgrounds, featuring interaction with NPCs through a dialogue system.

## Project Structure

### Core Game Folders

- **Rooms** - Game scenes including home, school, street, computer areas, and various mini-game levels
- **Objects** - Game entities including the player character (Henry), NPCs, buttons, walls, and interactive elements
- **Sprites** - Character animations, backgrounds, UI elements, and environmental assets
- **Scripts** - Game logic including dialogue topics, typing system, action handlers, and transitions
- **Sounds** - Audio assets for the game
- **Sequences** - Animation sequences including fade in/out and slide transitions
- **Fonts** - Custom fonts for menus, names, and dialogue text
- **Tile Sets** - Tileset definitions for various environments

### Mini-Games

The project includes several mini-games:

- **Flappybean** - Bird-flying style mini-game
- **Block Pushing** - Puzzle game involving pushing blocks
- **Platform** - Side-scrolling platformer sections
- **Mazes** - Maze navigation challenges

### Game Locations

- **Home** - Multiple home rooms with various scenes
- **School** - School building with multiple areas
- **Street** - Outdoor street areas
- **Computer** - Computer puzzle rooms
- **Hall** - School hallway sections
- **Playground** - Outdoor recreation area
- **Menu** - Main game menu and help screens

## Key Features

- **Dialogue System** - Topic-based conversation system with multiple NPCs
- **Character Animations** - Multi-directional movement animations (up, down, left, right, idle, jump, run)
- **Room Transitions** - Smooth fade and slide transitions between areas
- **Interactive NPCs** - Characters including Henry (player), Mom, Teachers, Friends, and Passers-by
- **Puzzle Elements** - Block pushing, maze navigation, and other interactive challenges
- **Text Box System** - Custom dialogue and text rendering

## Technical Details

- **Engine** - GameMaker Studio 2 (IDE Version 2024.8.1. 171)
- **Project File** - Spirit. yyp
- **Default Script Type** - GML (GameMaker Language)
